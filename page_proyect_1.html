Descripción técnica del proyecto

La realización de este formulario se llevó a cabo utilizando 4 tecnologías para el desarrollo de aplicaciones web. Con HTML se realizó la maquetación del formulario. Usando CSS se le dio un aspecto llamativo. Para lograr los efectos animados, los cuales permiten que cuando el usuario seleccione las opciones de pago, los campos necesarios aparecen y los que no son relevantes desaparecen. Ejemplo: Cuando el usuario selecciona la opción de realizar un pago vía PAYPAL, automáticamente aparece solo el campo requerido para realizar una transferencia desde esta plataforma, por lo tanto, solo se activa el campo en cual se introduce el correo electrónico. Este proceso se llevó a cabo usando la librería JQUERY. Para la validación de los datos del formulario se utilizó JAVASCRIPT puro, el cual permite verificar si los datos introducidos por el usuario son correctos o no, como, por ejemplo: si el usuario en el campo donde debe ir el nombre, este introduce un valor numérico. Saltará un mensaje de alerta que le dice al usuario, que los datos introducidos no son los esperados. Asimismo ocurre si el usuario deja alguno de los campos vacíos y esto son requeridos para enviar los datos al servidor.
