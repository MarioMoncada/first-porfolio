Descripción técnica del proyecto

Sitio web de una marca de ropa que está en proceso de creación. La finalidad de este sitio, es proporcionar a los clientes, información específica de cada uno sus productos, ofreciéndoles la facilidad de realizar compras en línea. Este proyecto se está elaborando con ReactJS el cual permite crear aplicaciones web muy rápidas.
