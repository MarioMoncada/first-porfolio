Proyecto Web
Café Plaza

Pagina de presentacion de un establecimiento comercial
que brinda información sobre sus productos.

Descripción técnica del proyecto

Este proyecto es un sitio web, que se creó con el fin de informar a las personas, sobre los productos gastronómicos que ofrece esta empresa. Para la realización de este página web, se utilizaron tecnologías tales como: HTML, CSS, JAVASCRITP y JQUERY .
El diseño de este sitio, es un diseño sencillo algo minimalista, que busca brindar a los usuarios una experiencia cómoda, proporcionándoles información puntual de los productos gastronómicos que allí podrán encontrar.
